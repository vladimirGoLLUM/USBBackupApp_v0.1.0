# USB Backup App — инструкция для пользователя

## Что это за программа

**USB Backup App** сравнивает содержимое выбранного USB‑накопителя (или диска) с папкой резервной копии на компьютере и помогает выполнить нужное действие: полный бэкап на диск, обновление папки бэкапа с флешки или перенос отличий обратно на флешку. Весь прогресс и журнал отображаются в одном окне, без лишних всплывающих окон во время работы.

## Системные требования

- **ОС:** Windows 10 или Windows 11 (64‑бит рекомендуется).
- **Права администратора:** обязательны только для настройки **автозапуска по флешке** (через Планировщик заданий). Обычное копирование и сканирование работают без повышения прав.
- **Интернет:** не требуется.
- **Python:** не нужен — используйте готовый `USBBackupApp.exe`.

## Установка (опционально: Inno Setup)

Стандартная поставка — **ZIP‑архив** с портативным `USBBackupApp.exe`. Распакуйте архив в любую папку и запустите exe.

Если вы собираете **установщик** сами (для IT‑отдела или маркетплейса), установите [Inno Setup](https://jrsoftware.org/isinfo.php) и выполните в корне репозитория разработчика:

```text
ISCC.exe USBBackupApp.iss
```

Готовый установщик появится в папке `installer_output\` (см. файл `USBBackupApp.iss`). Конечному пользователю достаточно ZIP из раздела «Готовый архив для продажи» в документации разработчика.

## Пошагово: скачал → запустил → настроил → готово

1. **Скачайте** архив `USBBackupApp_Sale_vX.Y.zip` и распакуйте в удобное место (например `Документы\USBBackupApp`).
2. **Запустите** `USBBackupApp.exe`. При первом запуске Windows SmartScreen может предупредить о неизвестном издателе — нажмите «Подробнее» → «Выполнить в любом случае», если доверяете источнику файла.
3. **Настройте:**
   - в блоке **Источник** выберите флешку (при необходимости нажмите **Обновить**);
   - в блоке **Настройки** укажите **Папку для бэкапа** (кнопка **Обзор**);
   - нажмите **Сканировать папку** и дождитесь окончания анализа;
   - нажмите предложенную кнопку режима (**СДЕЛАТЬ ПОЛНЫЙ БЭКАП**, **ОБНОВИТЬ ПАПКУ БЭКАПА** или **ЗАГРУЗИТЬ ОТЛИЧИЯ НА ФЛЕШКУ**).
4. **Готово:** дождитесь завершения копирования; при необходимости операцию можно **остановить** кнопкой в нижней панели.

Папка с настройками программы в режиме exe: файл состояния создаётся автоматически в `%APPDATA%\USBBackupApp\app_state.json` (запоминается последняя папка бэкапа). Шаблон без личных данных — в архиве: `config\app_state.json.example`.

## FAQ

### Что делать, если программа не запускается?

Проверьте SmartScreen и антивирус. Запустите exe из распакованной папки, а не из архива. Убедитесь, что используется Windows 10/11.

### Что делать, если сканирование «долго»?

После обхода файлов идёт **анализ отличий** — это нормально. Следите за текстом в нижней панели.

### Что делать, если часть файлов не скопировалась?

Иногда файлы заняты другим приложением или синхронизируются (например OneDrive). В конце операции может появиться предупреждение со списком пропущенных путей — закройте программы, блокирующие файлы, и повторите действие.

### Как устроен автозапуск (Планировщик и WMI)

Сейчас используются **оба** механизма, но роли разные:

- **Планировщик заданий Windows** — при входе пользователя в систему запускается фоновый процесс приложения (режим `--wmi-daemon`). Без этой задачи демон не стартует сам по себе после перезагрузки.
- **WMI** — уже работающий демон подписывается на события подключения томов и, когда вставлена **нужная** флешка (по серийному номеру тома), открывает окно программы и запускает сканирование.

Имя задачи в Планировщике: **`USBBackupApp_WMI_Daemon`** (устаревшая задача с другим именем при включении автозапуска может быть удалена автоматически).

### Как включить автозапуск?

1. Подключите флешку и запустите **USB Backup App от имени администратора** (для создания задачи в Планировщике нужны права администратора).
2. В списке **Источник** выберите эту флешку.
3. В блоке **Автозапуск по флешке** нажмите **Включить автозапуск** и дождитесь сообщения об успехе.
4. **Отключите и снова подключите** флешку — должно открыться окно приложения и начаться сканирование (если в настройках уже указана папка бэкапа).

### Как отключить автозапуск?

1. Запустите **USB Backup App от имени администратора**.
2. В блоке **Автозапуск по флешке** нажмите **Отключить автозапуск**.
3. Дождитесь сообщения об успешном отключении.

**Вручную:** откройте **Планировщик заданий** (`taskschd.msc`) → библиотека планировщика → найдите задачу **`USBBackupApp_WMI_Daemon`** и удалите её.

### Нужен ли интернет?

Нет.

### Программа удаляет файлы с флешки?

Основной сценарий — **копирование и синхронизация** по выбранному режиму. Перед работой с важными данными сделайте тест на копии.
