# USB Backup App — пакет для передачи клиенту

## Что отдавать клиенту

- `USBBackupApp.exe` (портативный файл)
- `USER_GUIDE.md`
- `CHANGELOG.md`
- `EULA.txt`
- `LICENSE.txt`

Рекомендуется упаковать в zip: `USBBackupApp_vX.Y.Z.zip`.

## Как собрать релиз

Все команды выполняются из папки `source/`.

1. Обновите версию в файле `VERSION`.
2. Установите зависимости (если ещё не установлены):

```powershell
pip install -r requirements.txt
```

3. Соберите exe через spec-файл:

```powershell
pyinstaller --noconfirm USBBackupApp.spec
```

Готовый файл: `dist\USBBackupApp.exe`.

4. Скопируйте exe в папку `release\USBBackupApp_vX.Y.Z\`.

## Примечания

- Windows SmartScreen может предупреждать о запуске неподписанного exe.
- Для уменьшения предупреждений обычно используют Code Signing Certificate.
- Перед передачей клиенту убедитесь, что в финальном архиве нет диагностических логов (`*.log`) и файлов состояния (`app_state.json`).

## FAQ

**На каких версиях Windows работает программа?**
Работает на Windows 10 и Windows 11.

**Нужен ли интернет для работы?**
Нет. Программа работает локально, интернет не требуется.

**Что делает программа?**
Сравнивает флешку и папку бэкапа, затем предлагает нужное действие:
- СДЕЛАТЬ ПОЛНЫЙ БЭКАП
- ОБНОВИТЬ ПАПКУ БЭКАПА
- ЗАГРУЗИТЬ ОТЛИЧИЯ НА ФЛЕШКУ

Весь прогресс отображается в нижней панели главного окна без всплывающих окон.

**Можно ли выбрать любую папку для бэкапа?**
Да. Можно выбрать любую папку на компьютере.

**Что будет, если флешку переименовать?**
Автозапуск ориентируется на ID устройства, поэтому переименование не мешает работе.

**Нужны ли права администратора?**
Для обычного копирования — не обязательно.
Для настройки автозапуска — нужны права администратора.

**Программа удаляет файлы?**
Нет, основная логика — копирование и обновление данных. Перед использованием рекомендуется протестировать на тестовой папке.

**Что делать, если автозапуск не сработал сразу?**
Откройте программу от имени администратора и заново включите автозапуск для нужной флешки. Затем переподключите флешку.

**Где лежат логи?**
Журнал операций отображается в правой панели главного окна. Внешние лог-файлы в релизной версии не создаются.
